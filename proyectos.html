Proyectos

A continuación, se mostrarán seis proyectos realizados a lo largo largo de la carrera, realizados tanto individualmente como en equipo, que se han llevado acabo usando distintas herramientas y metodologías. Tanto el código de algunos de estos proyectos, como el de otros, se encuentra disponible en GitHub en caso de que se quieran consultar más detenidamente.

Repositorios Proyectos
Reproductor Multimedia

(Repositorio) Proyecto realizado para la asignatura de Multimedia, que consiste en una aplicación desarrollada en Android Studio que es capaz de reproducir y comprimir imágenes, audio y vídeo. El lenguaje empleado para la realización de esta práctica fue Java.

Análisis de sentimientos

(Repositorio) Proyecto realizado para la asignatura de Integración de Sistemas Informáticos, que consiste en una aplicación desarrollada en Python que mediante el uso de una API de series y películas analiza los tweets de personas relacionados con ese tema para sacar conclusiones acerca de los datos.

Diseño de Microservicios

(Repositorio) El objetivo principal del proyecto es desarrollar un sistema distribuido basado en microservicios. Para ello se tomará como modelo la popular plataforma de streaming NetFlix para crear una pequeña plataforma de streaming bajo demanda, utilizando algunos de los servicios disponibles en ZeroC ICE.

Tamagotchi

(Repositorio) El objetivo principal del proyecto es desarrollar un juego haciendo uso de C#. Esta aplicación incluye animaciones y distintos elementos de videojuegos.

Sitio web

(Sitio Web) Desarrollo con HTML y CSS de un sitio web para la asignatura de Comercio Electrónico.

SEED Labs

(Repositorio) El objetivo del proyecto SEED es desarrollar ejercicios prácticos de laboratorio para la enseñanza de la seguridad informática. Ejercicios relacionados con ataques TCP/IP, encriptación, firewalls y vulnerabilidades de Buffer Overflow.
